# Dashboard

This folder contains the Power BI dashboard for Mutual Fund Analytics.

## File

- bluestock_mf_dashboard.pbix

## Open Dashboard

1. Open Power BI Desktop.
2. Click Open Report.
3. Select `bluestock_mf_dashboard.pbix`.

## Dashboard Features

- Total AUM Analysis
- Fund House Performance
- SIP Inflows Trend
- Category-wise Analysis
- Risk Metrics
- Fund Comparison
